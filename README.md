
# Odin Recipes
## Description
This project is part of *The Odin Project - Foundations Course*.
The goal is to build a basic recipe website using only HTML.
The website contains a main index page with links to different recipe pages.
---
## Skills Demonstrated
* Basic HTML structure
* Working with text (headings, paragraphs)
* Creating lists (ordered and unordered)
* Adding links between pages
* Embedding images
* Writing clear Git commit messages
---
## Project Structure
* `index.html` → main page with links to recipes
* `recipes/` → folder containing individual recipe pages
* `images/` → folder for images used in the website
---
## What I Learned
* How to structure a simple website using HTML
* How to organize files and folders
* The importance of clean and readable code
