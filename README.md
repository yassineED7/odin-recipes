# Odin Recipes

## Description

This project is part of *The Odin Project - Foundations Course*.
The goal is to build a basic recipe website using **HTML and CSS**.
The website contains a main index page with links to different recipe pages, all styled for better readability and structure.

---

## Skills Demonstrated

- **Basic HTML structure:** Headings, paragraphs, and lists.
- **Navigation:** Linking between multiple pages and directory structures.
- **CSS Foundations:** Mastering the **Box Model** (padding, border, margin).
- **Layout Design:** Centering elements and using **CSS Grid** for basic positioning.
- **Version Control:** Writing clear commit messages and handling **Git Merge Conflicts**.

---

## Project Structure

- `index.html` → Main page with links to recipes.
- `recipes/` → Folder containing individual recipe pages (Omelette, Pancakes, etc.).
- `images/` → Folder for images used in the website.
- `styles.css` → External stylesheet for global site design.

---

## What I Learned

- How to use the **Box Model** to create predictable and clean layouts.
- The importance of a **CSS Reset** (`box-sizing: border-box`) to simplify layout calculations.
- How to troubleshoot and fix Git errors when local and remote branches diverge.
